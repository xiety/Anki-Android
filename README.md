# AnkiDroidMod

Quality of life modifications of AnkiDroid.

### Main screen

1. Detailed number of due cards (new/learn/review)

1. Number of reviews behind the limit

1. Grayed out expanded parent deck numbers

### Review screen

1. Card type color line

1. Large and duplicated buttons for left and right one-handed control

### Small Widget

1. Detailed number of due cards (new/learn/review)
